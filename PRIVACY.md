# GitHub PR Assistant — Privacy Policy

**Data stored locally:** Your Anthropic API key, selected model, and optional
custom prompt are stored only in your browser via browser.storage.local. They
are never sent to the developer or any third party other than Anthropic.

**Data transmitted:** When you click "Use Claude API", the diff of the GitHub
pull request you are viewing is sent to the Anthropic Claude API
(https://api.anthropic.com) to generate a PR title and description. No data is
sent at any other time, and the "Generate JSON" feature transmits nothing.

**No tracking:** This extension contains no telemetry, analytics, or tracking
of any kind. The developer does not collect, receive, or store any of your data.

**Third parties:** Data sent to Anthropic is subject to Anthropic's privacy
policy: https://www.anthropic.com/legal/privacy

This extension is open source.
